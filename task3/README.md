# Task 3: Exploratory Data Analysis (EDA) Project

## 📋 Project Overview

Comprehensive exploratory data analysis of Global YouTube Statistics dataset to uncover patterns, trends, and insights about the YouTube ecosystem. This project uses statistical summaries, visualizations, and correlation analysis to identify key influencing factors.

**Status:** 🟢 Available (Due: 22 Jul 2026 - 7 Days Left)

## 🎯 Objectives

- Analyze YouTube dataset to uncover patterns and trends
- Use statistical summaries and visualizations
- Identify correlations and key influencing factors
- Present insights in a structured report
- Develop analytical thinking and data exploration skills

## 🚀 Key Features

### Statistical Analysis

- ✅ Descriptive statistics for all variables
- ✅ Distribution analysis
- ✅ Outlier detection using IQR method
- ✅ Missing value assessment
- ✅ Data quality checks

### Correlation Analysis

- ✅ Pearson correlation coefficients
- ✅ Correlation heatmaps
- ✅ Bivariate relationships
- ✅ Key influencing factors identification

### Visualizations Created

1. **Subscribers Distribution** - Histogram and boxplot
2. **Top 20 YouTubers** - Horizontal bar chart
3. **Category Distribution** - Pie chart
4. **Country Distribution** - Bar chart
5. **Subscribers vs Video Views** - Scatter plot
6. **Correlation Heatmap** - Feature correlations
7. **Avg Subscribers by Category** - Horizontal bar chart
8. **Upload Count Distribution** - Histogram
9. **Channel Type Distribution** - Bar chart
10. **Channel Creation Timeline** - Line chart

## 📊 Dataset

**Source:** `Global YouTube Statistics.csv`

**Key Features:**

- `rank` - Channel ranking
- `Youtuber` - Channel name
- `subscribers` - Number of subscribers
- `video views` - Total video views
- `category` - Content category
- `Country` - Channel country
- `uploads` - Number of uploads
- `channel_type` - Type of channel
- `created_year` - Year channel was created
- And many more demographic and engagement metrics
